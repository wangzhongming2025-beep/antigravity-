# 🔑 如何获取 Google Gemini API Key

目前 Google 尚未发布 "Gemini 3 Pro"，最新的稳定版本是 **Gemini 1.5 Pro** 或 **Gemini 2.0 Flash**。以下是获取通用 API Key 的方法：

---

## 第一步：访问 Google AI Studio
需要魔法（科学上网），访问地址：
👉 [https://aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)

## 第二步：创建 API Key
1.  登录你的 Google 账号。
2.  点击左上角的 **"Create API key"** 按钮。
3.  选择 **"Create API key in new project"** (在之前的项目中创建)。

## 第三步：复制保存
1.  生成的 Key 以 `AIza` 开头。
2.  复制这个 Key，保存好。

---

## ⚠️ 注意事项
*   **网络要求**：使用 Gemini 必须全程开启魔法（美国节点最佳）。
*   **免费额度**：目前 Google AI Studio 对个人开发者提供免费额度，非常良心。
*   **模型选择**：建议在工具中选择 `gemini-1.5-pro`，它的中文理解和长文本能力极强。
