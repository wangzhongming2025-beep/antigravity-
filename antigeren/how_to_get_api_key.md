# 🔑 极简教程：如何获取 DeepSeek API Key

为了使用咱们实战营的“AI洗稿工厂”，你需要一个 DeepSeek 的 API Key。
别担心，这很简单，只需 3 步，成本不到 1 块钱就能用很久。

---

## 第一步：注册/登录开发者平台
打开浏览器，访问 DeepSeek 开放平台：
👉 [https://platform.deepseek.com/](https://platform.deepseek.com/)

使用手机号注册登录即可。

## 第二步：创建 API Key
1.  登录后，在左侧菜单栏找到 **【API keys】** (像钥匙一样的图标)。
2.  点击右上角的蓝色按钮 **【+ Create API Key】**。
3.  在弹出的框里随便输入一个名字（比如：`xiaohongshu`），点击确认。

![API Guide](/Users/wangzhongming/.gemini/antigravity/brain/7c22f518-c453-4c96-9084-263e751a5867/deepseek_api_guide_screenshot_1768530782702.png)

## 第三步：复制并保存
1.  系统会弹出一串以 `sk-` 开头的长字符串，这就是你的 **API Key**。
2.  **点击复制**，然后立刻把它粘贴到备忘录里保存好。（注意：这个Key只显示一次，关掉窗口就看不到了！）

## 💡 常见问题 (FAQ)

**Q: 需要花钱吗？**
A: 新用户注册通常会送 **500万 token** (足够你写完整个训练营的作业了)。如果送的额度用完了，充值 5-10 块钱人民币，也够用几个月。DeepSeek 非常非常便宜，被称为“价格屠夫”。

**Q: 安全吗？**
A: 咱们的工具是“本地运行”的，你的 Key 只保存在你自己浏览器的缓存里，不会上传到我的服务器，非常安全。

**Q: 为什么生成的慢？**
A: 如果遇到生成慢，可能是 DeepSeek 官方服务器繁忙，稍等几分钟再试即可。
